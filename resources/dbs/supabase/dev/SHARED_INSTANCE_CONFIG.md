## Management Commands

From this directory (`resources/dbs/supabase/local`):

- Start: `supabase start`
- Stop: `supabase stop`
- Status: `supabase status`
- Reset: `supabase db reset`

## Notes

- This instance is shared across multiple projects
- Database changes should be coordinated between projects
- Consider using separate schemas for different projects if needed
